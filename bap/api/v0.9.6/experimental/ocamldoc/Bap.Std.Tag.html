val register : string -> ('a -> Core_kernel.Std.Sexp.t) -> 'a t
register name sexp creates a new variant constructor, i.e., a new branch in a variant type. This function has no side-effects, it just returns a witness of a new constructor, that can be later used for storing into value and extracting from it. This witness should be shared between user and creator of the value
val create : 'a t -> 'a -> value
create cons x creates a value using constructor cons and argument x
val value : 'a t -> value -> 'a option
value cons extracts a value associated with a constructor cons (Essentially, performs a pattern match on the specified variant branch)
val is : 'a t -> value -> bool
is cons v true if value v was constructed with constructor cons
val tagname : value -> string
tagname value returns a constructor name of the value
val name : 'a t -> string
name cons returns a name of a constructor.
