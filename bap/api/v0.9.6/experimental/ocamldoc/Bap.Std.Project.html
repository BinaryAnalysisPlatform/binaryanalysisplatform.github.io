all string tags supports the following substitutions:

  • $region_ - name of region of file to which it belongs. For example, in ELF this name will correspond to the section name
  • $symbol_ - name or address of the symbol to which this memory belongs
  • $asm - assembler listing of the memory region
  • $bil - BIL code of the tagged memory region
  • $block_ - name or address of a basic block to which this region belongs
  • $min_addr, $addr - starting address of a memory region
  • $max_addr - address of the last byte of a memory region.
