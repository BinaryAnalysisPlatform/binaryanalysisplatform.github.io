       ?image:Bap.Std.image ->
       ?sym:string -> Bap.Std.mem -> Bap.Std.block -> Bap.Std.abi
creates a set of ABI for the provided symbol. Until all parameter is set to true the ABI will be disambiguated, using choose method. Only equally valid ABI are returned.
val merge : Bap.Std.abi list -> Bap.Std.abi
merge abis create an abi that tries to take best from all provided abi. If the input list is empty, then the stub abi will be returned.
