Module Bap.Std.Disasm_expert.Recursive

module Recursive: sig .. end
Recursive Descent Disassembler. This disassembler is built on top of Basic disassembler. It uses work list algorithm to implement recursive descent disassembly and lazily reconstructs the whole program CFG.

This is an expert-level module, and it is suggested to use high-level Disasm interface, that is built ontop of this module.
