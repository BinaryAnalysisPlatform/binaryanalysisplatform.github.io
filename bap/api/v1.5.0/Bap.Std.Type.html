Module Bap.Std.Type

module Type: sig .. end
The type of a BIL expression.

Each BIL expression is either an immediate value of a given width, or a chunk of memory of a give size. The following predefined constructors are brought to the scope:

  • bool_t;
  • reg8_t;
  • reg16_t;
  • reg32_t;
  • reg64_t;
  • reg128_t;
  • reg256_t;
  • mem32_t;
  • mem64_t.

type t = Std.Bil.typ =
| Imm of int
| Mem of Std.addr_size * Std.size
type is either an immediate value or a storage
