NAME

emit-ida-script - extract a IDA python script from bap  

SYNOPSIS

emit-ida-script [OPTION]...  

DESCRIPTION

Iterates through memory for tagged BIR attributes, and dumps them into a python script, that can be later loaded into IDA. The special `color' tag causes the respective address to be colored.  

OPTIONS

--attr=VAL
Emit specified BIR attribute. Can be specified multiple times.
--file=NAME
Dump annotations to the specified file NAME. If not specified, then the script will dumped into the standard output
--help[=FMT] (default=auto)
Show this help in format FMT. The value FMT must be one of `auto', `pager', `groff' or `plain'. With `auto', the format is `pager` or `plain' whenever the TERM env var is `dumb' or undefined.
