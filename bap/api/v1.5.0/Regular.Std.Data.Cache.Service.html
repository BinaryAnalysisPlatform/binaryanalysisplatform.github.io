Module Regular.Std.Data.Cache.Service

module Service: sig .. end
Service injection point.

By default the library doesn't provide any caching services. All data stored in the cache is forgotten.

The caching service is provided by plugins, who register itself using Service.provide function. Only one caching service can be active at given moment.

Note: this interface is for implementing caching for data type. If you're using data type whose module T implements interface Std.Data, then you can just use T.Data.Cache module. If you need to add caching to your custom data type, then use Regular, Std.Opaque or Std.Data.Make functors (in the order of preference).

val provide : Std.Data.Cache.service -> unit
provide service will substitute current caching service with a new service.
val request : 'a Std.reader -> 'a Std.writer -> 'a Std.Data.Cache.t
request reader writer returns a caching service for type 'a.
