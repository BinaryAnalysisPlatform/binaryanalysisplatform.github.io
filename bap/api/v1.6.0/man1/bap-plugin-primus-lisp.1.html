NAME

primus-lisp - install and load Primus lisp libraries  

SYNOPSIS

primus-lisp [OPTION]...  

DESCRIPTION

Installs and registers the Primus Lisp library. The library implements stdlib interface in Lisp. Only init module is loaded automatically.

The plugin also provides an interface through which it is possible to load new libraries and modules, or to load existing modules.  

OPTIONS

--add=VAL
paths to lisp libraries
--channel-redirect=VAL
establishes a redirection between an emulated file path and a file path on a host system. Each redirection should be of form <emu-name>:<real-name>, where <emu-name> could be a path or a a name of one of the standard channels, i.e., <stdin> or <stdout> or <stderr>.
--documentation
outputs Primus Lisp documentation
--dump
dumps generated AST
--help[=FMT] (default=auto)
Show this help in format FMT. The value FMT must be one of `auto', `pager', `groff' or `plain'. With `auto', the format is `pager` or `plain' whenever the TERM env var is `dumb' or undefined.
--load=VAL (absent=posix)
load specified module
