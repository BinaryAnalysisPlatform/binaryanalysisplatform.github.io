callgraph-collator

Section: BAP Programmer's Manual (3)
Updated:
Index Return to Main Contents
 

DESCRIPTION

Compares projects by their callgraphs. Outputs differences immitating uniform diff format, e.g., 


- src -> dst

indicates that in the alternative version there is no call from src to dst, and 


+ src -> dst

means that in the base there is no call from src to dst.
