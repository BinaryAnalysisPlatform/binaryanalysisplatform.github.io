callsites

Section: BAP Programmer's Manual (3)
Updated:
Index Return to Main Contents
 

DESCRIPTION

This pass will inject artificial definitions of a subroutine arguments at call sites. Consider function malloc that has the following declaration in BIR: 



      sub malloc(malloc_size, malloc_result)
      malloc_size :: in u32 = R0
      malloc_result :: out u32 = R0

This plugin will add two definitions, one just before the call to the malloc: 



      ...
      000001c3: malloc_size := R0
      0000015b: call @malloc with return %0000015c

And prepend another to the block to which malloc will return: 



      0000015c:
      000001c4: R0 := malloc_result
      ...
