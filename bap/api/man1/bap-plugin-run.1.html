run

Section: BAP Programmer's Manual (3)
Updated:
Index Return to Main Contents
 

DESCRIPTION

Populates and run the Primus Jobs queue. Creates new job for each specified system and runs the queued jobs (including those that were already added to the queue by other analyses). After the job queue is emptied, commits the obtained knowledge and passes the resulting project data structure downstream.
