radare2

Section: BAP Programmer's Manual (3)
Updated:
Index Return to Main Contents
 

DESCRIPTION

This plugin provides a symbolizer based on radare2.  

EXAMPLES

To view the symbols after running the plugin:

bap executable --dump-symbols

To view symbols without this plugin:

bap executable --no-radare2 --dump-symbols
