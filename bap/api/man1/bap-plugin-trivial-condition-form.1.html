trivial-condition-form

Section: BAP Programmer's Manual (3)
Updated:
Index Return to Main Contents
 

DESCRIPTION

Ensures that all branching conditions are either a variable or a constant. We call such representation a Trivial Condition Form (TCF). During the translation all complex condition expressions are hoisted into the assignment section of a block.
