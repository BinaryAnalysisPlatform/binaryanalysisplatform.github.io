relocatable

Section: BAP Programmer's Manual (3)
Updated:
Index Return to Main Contents
 

DESCRIPTION

Extracts symbolic information from the program relocations.

The relocation symbolizer leverages the relocation information stored in files to extract symbol names. Since a relocation references an external symbol which doesn't have an address we use an address of a callsite.
