frontc-parser

Section: BAP Programmer's Manual (3)
Updated:
Index Return to Main Contents
 

DESCRIPTION

A parser for C header files, that uses FrontC library as a backend. The parser understands GNU attribute syntax, that can be used to pass arbitrary semantic information in the header.
