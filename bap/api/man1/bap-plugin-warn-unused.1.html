warn-unused

Section: BAP Programmer's Manual (3)
Updated:
Index Return to Main Contents
 

SYNOPSIS



    --warn-unused
    --warn-unused-taint
    --warn-unused-print
    --warn-unused-mark
 

DESCRIPTION

If a subroutine has GNU attribute warn_unused_result and its result is not used, then print a warning message.  

PASSES

--warn-unused-taint Taint all values defined by functions that are marked with warn_unused_result attribute. Will run callsites as a dependency.

--warn-unused-print Print all calls that weren't checked.

--warn-unused-mark Mark all unchecked calls with Term.dead attribute

--warn-unused Same as --warn-unused-taint --propagate-taint --warn-unused-print
