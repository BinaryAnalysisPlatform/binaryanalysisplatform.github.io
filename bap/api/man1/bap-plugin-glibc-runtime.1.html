glibc-runtime

Section: BAP Programmer's Manual (3)
Updated:
Index Return to Main Contents

Enables ad-hoc support for glibc runtime code. In particular it detects the locations of main and b(,__libc_start_main) functions (and adds the latter if it is absent).
