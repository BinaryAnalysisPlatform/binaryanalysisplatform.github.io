primus-exploring

Section: BAP Programmer's Manual (3)
Updated:
Index Return to Main Contents
 

DESCRIPTION

The exploring scheduler will prioritize clones that will wonder into not yet explored or less explored areas. More specifically, from a set of machine clones, it will choose those, that will proceed to a basic block that was visited the least amount of times. The round-robin scheduler will switch the context after each basic block. It will count the number of time the block was evaluated
