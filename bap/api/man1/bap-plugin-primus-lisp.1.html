primus-lisp

Section: BAP Programmer's Manual (3)
Updated:
Index Return to Main Contents
 

DESCRIPTION

Installs and registers the Primus Lisp library. The library implements stdlib interface in Lisp. Only init module is loaded automatically.

The plugin also provides an interface through which it is possible to load new libraries and modules, or to load existing modules.
