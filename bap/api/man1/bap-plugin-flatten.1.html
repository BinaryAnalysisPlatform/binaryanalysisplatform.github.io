flatten

Section: BAP Programmer's Manual (3)
Updated:
Index Return to Main Contents
 

DESCRIPTION

Flatten all AST in the program.  

EXAMPLE



  ;; input 
  #10 := 11 * (#9 + 13) - 17
  ;; output
  #11 := #9 + 13
  #12 := 11 * #11 
  #10 := #12 - 17
