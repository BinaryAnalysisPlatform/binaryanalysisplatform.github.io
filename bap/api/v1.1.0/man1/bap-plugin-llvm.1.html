NAME

llvm - provide loader and disassembler using LLVM library  

SYNOPSIS

llvm [OPTION]...  

DESCRIPTION

Uses LLVM library to provide disassembler and file loader  

OPTIONS

--help[=FMT] (default=pager)
Show this help in format FMT (pager, plain or groff).
--x86-syntax=VAL (absent=att)
Choose style of code for x86 syntax between either `att' or `intel'
