Module Bap.Std.Var

BIL variable. BIL variables are regular values. Every call to create will create a fresh variable.
type t = var 
include Bap.Std.Regular
implements Regular interface
val create : ?tmp:bool -> string -> typ -> t
create ?tmp name typ creates a fresh new variable with assosiated name and type typ. The created variable is absolutely new, and is comparable to true only with itself.
tmp : designates variable is temporary, with whatever meaning assosiated to it by a caller.
val name : t -> string
val typ : t -> typ
val is_tmp : t -> bool
is_tmp true if variable is temporary
