Module Bap.Std.Project.Info

module Info: sig .. end
Information obtained during project reconstruction.

These pieces of information are guaranteed to be discovered during the project reconstruction. See Std.Project.create function for more information on the reconstruction process.

val file : string Bap_future.Std.stream
occurs everytime a new file is opened. The value is a filename
val arch : Std.arch Bap_future.Std.stream
occurs once input architecture is known
val data : Std.value Std.memmap Bap_future.Std.stream
occurs once input memory is loaded
val code : Std.value Std.memmap Bap_future.Std.stream
occurs once code segment is discovered
val cfg : Std.cfg Bap_future.Std.stream
occurs everytime a whole program control flow graph is changed
val symtab : Std.symtab Bap_future.Std.stream
occurs everytime a symbol table is changed
val program : Std.program Std.term Bap_future.Std.stream
occurs every time a program term is changed during the project reconstruction process.
val spec : Ogre.Doc.t Bap_future.Std.stream
occurs once image spec is known
