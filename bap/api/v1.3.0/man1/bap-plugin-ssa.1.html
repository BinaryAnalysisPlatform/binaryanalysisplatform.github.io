NAME

ssa - translates a program into the SSA form  

SYNOPSIS


    ssa
 

DESCRIPTION

Translates the whole program into the SSA form  

OPTIONS

--help[=FMT] (default=pager)
Show this help in format FMT (pager, plain or groff).
