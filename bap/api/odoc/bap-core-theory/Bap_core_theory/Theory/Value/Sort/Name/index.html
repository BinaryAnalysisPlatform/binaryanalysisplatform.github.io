Module Sort.Name

The name registry.

Names of symbols must be unique as the name is used as a witness of authenticity of the sort. Once obtained, the name should be kept secret beyond the module signature.

See the Example section of the Sort module documentation for more information.

type t
val declare : ?⁠package:string -> string -> name

declare ?package name declares a new name.

The declared name must be unique to the package. If such name is already declared in the package, then the declaration fails.
