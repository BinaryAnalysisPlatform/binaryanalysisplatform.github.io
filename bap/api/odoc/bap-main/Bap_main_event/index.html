val register_printer : (t -> string option) -> unit

register_printer f when the event e is printed, f e must be None if f is not a subset of events, that is intended to be printed by f. If it is Some str, then str is printed out.

If more than one printer returns Some thing for the same event, then the last registered has the precedence.
