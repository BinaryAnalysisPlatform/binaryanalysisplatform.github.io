val lift_c : src:Bap.Std.exp -> Arm_types.shift -> shift:Bap.Std.exp -> Bap.Std.typ -> Bap.Std.exp * Bap.Std.exp

Need the operand and the carry flag value src - the source, if you intend to use the carry bit, this must not be the destination of the shift expression. This means it must be a temp that contains the value of the source not the actual source itself shift_type - the type of shift shift - must be a exp that is the amount of the shift (ignored for rrx)

decodes a register shifted operand * src - the operand to be shifted, cannot be the destination * in practice this means it must be a temp variable. * shift_type - an int64, bits 2 through 0 represent the shift type * valid shift types are number 1 through 5 * shift - the value to shift by * t - the type *
