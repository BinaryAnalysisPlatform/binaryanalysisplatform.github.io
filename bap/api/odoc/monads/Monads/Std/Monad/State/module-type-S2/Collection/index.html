Module S2.Collection

Lifts collection interface into the monad.

This module provides two functors that take a basic collection interface and provide the Monad.Collection.S2 interface.

See Monad.Collection for more information.

module type S = Collection.S2 with type ('a, 'e) m := ('a, 'e) t
module Eager : functor (T : Collection.Eager) -> S with type 'a t := 'a T.t

Eager(C) derives Monad.Collection.S2 for container C

module Delay : functor (T : Collection.Delay) -> S with type 'a t := 'a T.t

Delay(C) derives Monad.Collection.S2 for container C
