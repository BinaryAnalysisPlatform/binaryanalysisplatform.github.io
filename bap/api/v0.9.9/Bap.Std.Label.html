Module Bap.Std.Label

module Label: sig .. end
Target of a control flow transfer.

Labels can be direct, that are known to us. Or indirect, that are arbitrary expressions.
type t = label 
val create : unit -> t
create () creates a new label with a freshly generated identifier.
val direct : tid -> t
direct label creates a direct label with a given identifier.
val indirect : exp -> t
indirect exp creates a label that is resolved to an expression exp
val change : ?direct:(tid -> tid) ->
       ?indirect:(exp -> exp) -> t -> t
updates the label
