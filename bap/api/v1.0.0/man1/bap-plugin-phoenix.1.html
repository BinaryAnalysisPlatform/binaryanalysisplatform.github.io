NAME

phoenix - output project information in a phoenix format  

SYNOPSIS

phoenix [OPTION]...  

DESCRIPTION

Output information about processed binary in a human readable format. This will emit CFG for each format in dot format. It will also store BIL and ASM code in html format.Output folder can be optionally specified. If omitted, the basename of the target file will be used as a directory name.  

OPTIONS

--help[=FMT] (default=pager)
Show this help in format FMT (pager, plain or groff).
--keep-alive
Keep alive unused temporary variables
--keep-const
Disable constant folding
--labels-with=VAL (absent=name)
Put block name, assembler instructions, or bil instructions on graph labels using `name', `asm', or `bil' respectively. Can be specified as a list of multiple elements separated by commas.
--labels-with-asm
DEPRECATED. Put assembler instructions on graph labels
--labels-with-bil
DEPRECATED. Put bil instructions on graph labels
--labels-with-name
DEPRECATED. Put block name on graph labels
--no-inline
Disable inlining temporary variables
--no-optimizations
Disable all kinds of optimizations
--no-resolve
Do not resolve addresses to symbolic names
--output-folder=VAL (absent=phoenix)
Output data into the specified folder
