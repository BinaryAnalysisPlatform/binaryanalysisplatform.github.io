Module Bap.Std.Disasm

module Disasm: sig .. end
Disassembled program.

This module provides an interface for values of type disasm.

type t = disasm 
val insns : t -> (mem * insn) seq
returns all instructions that was successfully decoded in an ascending order of their addresses. Each instruction is accompanied with its block of memory.
val blocks : t -> block table
returns a mapping from memory regions to basic blocks. One may think this as a whole program CFG that is constructed on the fly.
val insn_at_mem : t -> mem -> insn option
performs lookup for an instruction that occupies exactly the given piece of memory. If you need to find all instructions that lies in a given region of memory, use insns_of_mem or insns_of_block functions.
val insns_at_mem : t -> mem -> (mem * insn) seq
insns_of_mem returns all instructions that occupies memory regions that have intersections with mem.
val mems_of_insn : t -> insn -> mem seq
returns a sequence of memory regions occupied by the given instruction.
val insn_at_addr : t -> addr -> (mem * insn) option
insn_at_addr t addr finds instruction to which the addr belongs. In other words if instruction at given address is found it doesn't mean, that it starts at this address. Consider comparison with min_addr if you need to match starting address only.
val span : t -> mem seq
returns a blocks of memory that was visited during the disassembly. The regions are merged with Memory.merge if possible. So it returns the least possible amount of contiguous memory regions
type error = [ `Failed of Error.t
       | `Failed_to_disasm of mem
       | `Failed_to_lift of mem * insn * Error.t ] 
module Error: Printable  with type t := error
val errors : t -> (mem * error) seq
returns a list of all errors and warnings that occurred during the disassembling
val block : addr tag
start of basic block
val insn : insn tag
machine instruction
val insn_addr : addr tag
address of instruction
