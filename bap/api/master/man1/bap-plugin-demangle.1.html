NAME

demangle - demangle subroutine names  

SYNOPSIS

demangle [OPTION]...  

DESCRIPTION

Apply specified demangler to all subroutine names  

OPTIONS

--help[=FMT] (default=pager)
Show this help in format FMT (pager, plain or groff).
--with=DEMANGLER
Demangle all function names using DEMANGLER
