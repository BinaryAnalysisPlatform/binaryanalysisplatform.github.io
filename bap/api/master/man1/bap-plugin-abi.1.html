NAME

abi - apply abi information to a project  

SYNOPSIS

abi [OPTION]...  

DESCRIPTION

Run ABI specific passes. The passes are registered using Bap_abi.register_pass function, by target specific plugins, depending on the binary architecture and command line options, provided by a user.  

OPTIONS

--help[=FMT] (default=pager)
Show this help in format FMT (pager, plain or groff).
 

SEE ALSO

bap-plugin-x86(1), bap-plugin-arm(1), bap-abi(3)
