NAME

byteweight - find function starts using Byteweight algorithm  

SYNOPSIS

byteweight [OPTION]...  

DESCRIPTION

This plugin will provide a rooter service. The algorithm is described in [1]. The SEE ALSO section contains links for other plugins, that provides rooters

[1]: Bao, Tiffany, et al. "Byteweight: Learning to recognize functions in binary code." 23rd USENIX Security Symposium (USENIX Security 14). 2014.  

OPTIONS

--help[=FMT] (default=pager)
Show this help in format FMT (pager, plain or groff).
--length=VAL (absent=16)
Maximum prefix length when byteweighting
--sigs=VAL
Path to the signature file. No needed by default, usually it is enough to run `bap-byteweight update'.
--threshold=VAL (absent=0.9)
Minimum score for the function start
