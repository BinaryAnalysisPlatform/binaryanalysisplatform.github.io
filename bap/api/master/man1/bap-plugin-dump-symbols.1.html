NAME

dump-symbols - dump symbol information as a list of blocks  

SYNOPSIS

dump-symbols [OPTION]...  

DESCRIPTION

Output symbol information. In the output file, each symbol is in format of: `(<symbol name> <symbol start address> <symbol end address>)', e.g., `(malloc 0x11034 0x11038)'  

OPTIONS

--file=FILE
Dump symbols to the specified FILE
--help[=FMT] (default=auto)
Show this help in format FMT. The value FMT must be one of `auto', `pager', `groff' or `plain'. With `auto', the format is `pager` or `plain' whenever the TERM env var is `dumb' or undefined.
