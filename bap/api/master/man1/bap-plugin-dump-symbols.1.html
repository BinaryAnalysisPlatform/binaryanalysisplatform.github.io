NAME

dump-symbols - dump symbol information as a list of blocks  

SYNOPSIS

dump-symbols [OPTION]...  

DESCRIPTION

Output symbol information. In the output file, each symbol is in format of: `(<symbol name> <symbol start address> <symbol end address>)', e.g., `(malloc 0x11034 0x11038)'  

OPTIONS

--file=FILE
Dump symbols to the specified FILE
--help[=FMT] (default=pager)
Show this help in format FMT (pager, plain or groff).
