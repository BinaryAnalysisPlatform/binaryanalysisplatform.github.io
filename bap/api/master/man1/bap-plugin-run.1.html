NAME

run - a pass that will run a program  

SYNOPSIS

run [OPTION]...  

DESCRIPTION

Run program in the Primus emulator.  

OPTIONS

--argv=VAL
Program command line arguments
--entry-point=VAL (absent=_start)
When specified, then start the execution from VAL, otherwise the execution will be started from a default entry point
--env=VAL
Program environemt as a list of VAR=VAL pairs
--help[=FMT] (default=pager)
Show this help in format FMT (pager, plain or groff).
