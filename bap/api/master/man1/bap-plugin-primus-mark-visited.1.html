NAME

primus-mark-visited - marks terms that were visited by Primus  

SYNOPSIS

primus-mark-visited [OPTION]...  

DESCRIPTION

Marks all terms visited by any Primus machine with the [Term.visited] attribute.  

OPTIONS

--help[=FMT] (default=pager)
Show this help in format FMT (pager, plain or groff).
