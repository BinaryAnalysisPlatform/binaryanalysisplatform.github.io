NAME

cxxfilt - provide c++filt based demangler  

SYNOPSIS

cxxfilt [OPTION]...  

DESCRIPTION

A demangler that relies on a c++filt utility  

OPTIONS

--help[=FMT] (default=pager)
Show this help in format FMT (pager, plain or groff).
