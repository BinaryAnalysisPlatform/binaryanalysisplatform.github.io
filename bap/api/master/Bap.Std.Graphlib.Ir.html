Module Bap.Std.Graphlib.Ir

module Ir: sig .. end
IR code as a graph.

This module implements a graph view on the IR representation of a program. The underlying implementation uses a value of type sub term to implement most of the operations. But in order to get fast bidirectional access to a graph, it augments term with extra information, like back edges.
