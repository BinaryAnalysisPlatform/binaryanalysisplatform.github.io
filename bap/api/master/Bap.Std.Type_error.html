Module Bap.Std.Type_error

module Type_error: sig .. end
BIL type errors.

A type error will occur during the evaluation of a BIL program that is not well typed.

type t 
val bad_mem : t
bad_mem error occurs when a value of type mem was expected
val bad_imm : t
bad_imm error occurs when a value of type imm was expected
val bad_cast : t
bad_cast error occurs when parameters to the cast operation are not valid.
val bad_type : exp:typ -> got:typ -> t
bad_type ~exp ~got error happens when we expect a value of type exp but got a value of type got.
