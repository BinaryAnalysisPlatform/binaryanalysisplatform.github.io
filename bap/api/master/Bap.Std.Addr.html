Module Bap.Std.Addr

module Addr: sig .. end
Shortcut for bitvectors that represent addresses
include Bap.Std.Bitvector
val memref : ?disp:int ->
       ?index:int -> ?scale:size -> addr -> addr
memref ?disp ?index ?scale base mimics a memory reference syntax in gas assembler, dis(base,index,scale) assembler operation. It returns address at base + index * scale + dis.
disp : stands for displacement and defaults to 0
index : defaults for 0
scale : defaults to `r8

All operations are taken modulo , where n = bitwidth base.
