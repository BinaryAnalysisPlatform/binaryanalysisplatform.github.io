Module Bap.Std.Dict

module Dict: sig .. end
Universal Heterogeneous Map.

The dictionary can store values of arbitrary type. Only one value of a given tag can be stored in the map. For example, if you have tag cconv (calling convention) then it is guaranteed that in map there is zero or one value with this tag.
