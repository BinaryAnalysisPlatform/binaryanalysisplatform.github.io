Module Bap.Std.Project

module Project: sig .. end
Target of analysis.

A project groups together all the information recovered from the underlying binary information. It is also used for interaction between plugins.

val substitute : ?tags:string tag list -> t -> t
substitute ?tags p apply substitutions in text values associated with the provided tags in project memory. tags defaults to [comment; python; shell].

The following substitutions are performed:

  • $region{_name,_addr,_min_addr,_max_addr} - name of region of file to which it belongs. For example, in ELF this name will correspond to the section name
  • $symbol{_name,_addr,_min_addr,_max_addr} - name or address of the symbol to which this memory belongs
  • $asm - assembler listing of the memory region
  • $bil - BIL code of the tagged memory region
  • $block{_name,_addr,_min_addr,_max_addr} - name or address of a basic block to which this region belongs
  • $min_addr, $addr - starting address of a memory region
  • $max_addr - address of the last byte of a memory region.
