Module Bap.Std.Blk.Builder

module Builder: sig .. end
Builder interface.

This interface provides an efficient way to build new blocks. It is also useful, when rebuilding existing block, as it allows to specify the tid of the block. It is a user responsibility to preserve the uniqueness of tidentifiers throughout the program instance.
type t 
val create : ?tid:Bap.Std.tid ->
       ?phis:int -> ?defs:int -> ?jmps:int -> unit -> t
create ~tid ~phis ~defs ~jmp () creates a block builder. If tid parameter is specified, then the new block will have this tid. If any of phis, defs or jmps parameters are specified, the provtided number would be used as a hint of the expected amount of the corresponding entries. Since it is the hint, it can mismatch with the actual size. The hint must be a positive number.
