Module Bap.Std.Program

module Program: sig .. end
Program.

Program is a collection of function terms.
type t = program term 
val create : unit -> t
create () creates an empty program.
val lift : addr list ->
       block table -> program term
lift roots cfg takes a set of function starts and a whole program cfg and returns a program term.
val lookup : ('a, 'b) cls ->
       t -> tid -> 'b term option
lookup t program id is like find but performs deep lookup in the whole program for a term with a given id. This function is memoized, so it has amortized O(1) complexity, with a wostcase complexity of $O(N)$, where $N$ is the total amount of terms in program.
val parent : ('a, 'b) cls ->
       t -> tid -> 'a term option
parent t program id is Some p iff find t p id <> None
